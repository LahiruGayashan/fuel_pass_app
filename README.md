# fuel_pass_app
The Fuel Pass QR System is a mobile and web-based solution that manages fuel distribution using QR codes. Each vehicle receives a unique QR linked to a weekly quota. Fuel stations scan the code to validate and issue fuel, ensuring fair usage, reducing queues, and preventing misuse through real-time tracking.

# Key Objectives

1 Ensure fair fuel distribution

2  Reduce waiting time and queues

3  Prevent illegal fuel usage

4  Digitize the fuel management system

5  Provide real-time monitoring and tracking
